Simple Task Master


A sleek, responsive To-Do List application built with vanilla JavaScript. Perfect for staying organized and keeping track of daily goals.

Features
Add Tasks: Quickly jot down new items.
Mark as Done: Toggle completion status with a single click.
Delete Tasks: Remove items you no longer need.
Local Storage: Your tasks persist even if you refresh the page (Note: Include this if you added LocalStorage functionality!).
Responsive Design: Works beautifully on desktops, tablets, and phones.

TechnologyUsage
HTML5 :Semantic structure and layout.
CSS3:Custom styling and Flexbox/Grid layouts.
JavaScript: DOM manipulation and task logic.
