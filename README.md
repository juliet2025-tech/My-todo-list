Simple Task Master

Simple Task Master is a sleek, responsive, and user-friendly To-Do List application built with vanilla JavaScript, HTML, and CSS. It helps users manage their daily tasks efficiently, avoid procrastination, and stay organized.

Features:

Add Tasks: Quickly add new tasks with details such as due date, reminder time, priority, category, and repetition.

Mark as Done: Toggle the completion status of tasks with a single click. Completed tasks are visually distinct with a strike-through and color indicator.

Edit Tasks: Easily update the task name or details after adding.

Delete Tasks: Remove tasks you no longer need.

Search Tasks: Find tasks quickly by typing keywords in the search bar.

Filter Tasks: View all tasks, only completed tasks, or only pending tasks.

Sort Tasks: Sort tasks by due date to prioritize effectively.

Clear Completed Tasks: Remove all completed tasks in one click.

Repetitive Tasks: Automatically generate daily or weekly recurring tasks.

Local Storage: All tasks persist across page reloads and browser sessions.

Responsive Design: Works seamlessly on desktops, tablets, and mobile devices.

Technologies Used:

HTML5: Semantic structure for accessibility and readability.

CSS3: Custom styling with Flexbox and Grid layouts for responsive design.

JavaScript (Vanilla): DOM manipulation, event handling, and task logic.

LocalStorage API: Persistent storage for tasks.

Browser Notifications API: Optional reminders for tasks with notifications.

How It Was Built:

Planning the Features: Defined the core functionalities—add, edit, delete, filter, and sort tasks.

HTML Structure: Created semantic markup for inputs, task list, and buttons.

CSS Styling: Built a clean, modern, and mobile-first design using Flexbox and Grid.

JavaScript Logic: Implemented task creation, completion toggling, editing, deletion, filtering, sorting, and local storage functionality.

Testing & Refinement: Ensured the app works across different browsers and devices, and added enhancements like repetitive tasks and notifications.

Usage:

Clone or Download the repository.

Open index.html in your browser.

Start Adding Tasks using the input fields.

Manage Your Tasks: Mark complete, edit, delete, search, filter, or sort tasks.
